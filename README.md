## Задача

Необходимо дописать функциональность в файл `src/fibonacci/__main__.py`, который вычисляет N-е число Фибоначчи.

### Требования

1. **Реализация функции вычисления числа Фибоначчи**:
    - Реализовать функцию, которая будет принимать на вход число N (целое число) и возвращать N-е число Фибоначчи.
    - Формат входных данных: целое число N (например, N = 5).
    - Если N отрицательное, возвращать HTTP-ошибку 400.
    - Формат выходных данных: объект с полями:
        - `status` - строка, указывающая на успешность выполнения запроса.
        - `fibonacci_number` - число Фибоначчи для переданного N.
    
    Пример:
    ```json
    {
      "status": "success",
      "fibonacci_number": 5
    }
    ```


2. **Тестирование**:
    - После реализации функции необходимо чтоб все тесты были выполнены успешно.
    - Использовать команду `make test` для запуска тестов.

3. **Запуск сервера**:
    - Для запуска сервера используется команда `make run`.


### Технические детали

- Все изменения должны быть выполнены в файле `src/fibonacci/__main__.py`.
- Для запуска и тестирования используйте команду `make test` (для тестов) и `make run` (для запуска сервера).
